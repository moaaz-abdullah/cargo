# 🚗 Cargo | E-Commerce Car Sales Platform

Cargo is a front-end **e-commerce car sales platform** built using **HTML, CSS, and JavaScript**.
It allows users to explore different car manufacturers, browse models, view specifications, and place orders.

---

## 🚀 Overview

The platform simulates a real-world car showroom experience online, where users can:

* Browse multiple **car manufacturers**
* Explore various **car models**
* View detailed **car specifications**
* Check **pricing information**
* Place orders directly through the website
* Access **dealership/contact information**

---

## 🧠 Features

* Multi-brand car browsing (4 manufacturers)
* Dynamic model selection and display
* Car specifications and pricing details
* Order placement functionality
* Clean and user-friendly interface
* Responsive design (if applied)
